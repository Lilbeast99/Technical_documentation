Introduction

PHP (a recursive initialism for PHP: Hypertext Preprocessor) is an open-source server-side scripting language that can be embedded into HTML to build web applications and dynamic websites.

To embedd php into your websites, we use the php tags in the body of your html document. The php code then goes in between the tags like shown below.

// start of PHP code
<?php
// PHP code goes here
?>
// end of PHP

Printing Stuff on Screen

To display things on screen, we use the "echo" command. We then enclose whatever it is we want to print on scrin in quotes.

<?php
echo "Hello World!";
?>

We can also use the print function to display items out on screen in php. Though it is not widely used. The ptint function works like shown below.

<?php
print "Hello World!";
?>

Php Variables

Variables are containers allow us to store data, which we can then reference in our programs To declare a variable in php, we use the "$" sign, followed by the variable name then we equate them to the variable value like shown below.

<?php
$my_var = 21;
?>

After declaring a variable, we can then embed it in our program by by referencing the dollar-sign "$" followed by the variable name we used.

<?php
echo "MY age is $my_var"
?>

This will print out

"My age is 21"

Comments

Comments in php, allow us to give description to our lines of code. This is because comments are not executed by the IDE. In php, there are 2 types of comments: single-line comments, and multi-lines comments.

Single line comments, comment out just a single line. Single line comments are used as shown below

// This is a single line comment

Multi-line comments allow us to comment out multiple lines of code in php. Multi-line comments in php are used as shown below.

/*
This block of code
will not be executed
as it has been commented out
*/

Data-Types

PHP has support for some major datatypes. This include:

  • String
  • Integers
  • Boolean
  • Float
  • Special symbols

$phrase = "This is a string data-type"
$AGE = 21 // This is an integer data-type
$Fact = True/False // This is a boolean data-type
$Height = 5.8 // This is a float data-type

The data-types above, are all of the data-types supported within the PHP language. Special symbols like underscore "_", Commas ",", ecxlamation marks "!" etc. are also supported in the PHP language.
